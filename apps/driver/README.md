# كيات السائق

تطبيق Flutter خفيف للسائقين ضمن مشروع كيات. السائق يختار الخط، يدخل اسم/رقم الكية، ثم يشغل التتبع الحي حتى يظهر للركاب.

## التشغيل المحلي

```bash
cd apps/driver
flutter pub get
flutter run --dart-define=API_URL=http://localhost:3000
```

إذا كنت تشغل التطبيق على جهاز حقيقي، استخدم عنوان الـ IP المحلي بدلاً من `localhost`.

لخرائط iOS، انسخ ملف الأسرار وأضف مفتاح Google Maps:

```bash
cp ios/Flutter/Secrets.xcconfig.example ios/Flutter/Secrets.xcconfig
```

## ما يغطيه التطبيق

- دخول السائق برقم الهاتف و OTP. في التطوير الرمز الثابت هو `123456`.
- جلب خطوط كيات من backend.
- حفظ آخر اسم/رقم كية على الجهاز.
- إنشاء مركبة للسائق على الخط المختار.
- إرسال الموقع الحي مع السرعة عند توفرها.
- عرض خريطة فيها موقع الكية، مسار الخط، وركاب الانتظار.
- عرض ركاب الانتظار النشطين على نفس الخط.
- إيقاف التتبع من التطبيق والسيرفر.
